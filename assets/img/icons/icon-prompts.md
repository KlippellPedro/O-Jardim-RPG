# Catalogo de icones para gerar

Use este prompt base em todos os pedidos ao GPT-4o e adicione a descricao especifica de cada icone.

## Prompt base

Crie um icone 2D para um sistema de RPG dark fantasy chamado "O Jardim RPG". O icone deve ser bonito, limpo e legivel em tamanho pequeno, com fundo transparente, composicao centralizada, acabamento de metal escuro e brilho magico sutil. Nao incluir texto, letras, numeros, moldura quadrada, emoji literal nem fundo. Estilo: fantasia sombria elegante, simbolo premium de interface, 512x512 px.
